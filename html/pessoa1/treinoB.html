Agachamento Smith

  • Posicione-se sob a barra do equipamento Smith com os pés na largura dos ombros.
  • Desça lentamente até que suas coxas fiquem paralelas ao chão.
  • Empurre para cima retornando à posição inicial.
  • Mantenha as costas retas durante todo o movimento.
  • Repita para o número de repetições desejado.
×

Leg Press 45º

  • Sente-se no equipamento com os pés posicionados na plataforma na largura dos ombros.
  • Empurre a plataforma para cima até que as pernas estejam quase estendidas.
  • Baixe lentamente a plataforma até que os joelhos formem um ângulo de 90 graus.
  • Mantenha os pés firmes na plataforma durante o movimento.
  • Repita para o número de repetições desejado.
×

Banco Extensor

  • Sente-se no equipamento com as costas apoiadas no encosto.
  • Posicione os pés sob o rolo acolchoado.
  • Estenda as pernas completamente, contraindo os músculos das coxas.
  • Retorne lentamente à posição inicial.
  • Repita para o número de repetições desejado.
×

Leg Curl

  • Deite-se no equipamento com os tornozelos posicionados sob o rolo acolchoado.
  • Flexione os joelhos, puxando o rolo em direção aos glúteos.
  • Retorne lentamente à posição inicial.
  • Mantenha os quadris fixos durante o movimento.
  • Repita para o número de repetições desejado.
×

Panturrilha Sentado

  • Sente-se no equipamento com os pés apoiados na plataforma e os joelhos sob as almofadas.
  • Levante os calcanhares o máximo possível, contraindo as panturrilhas.
  • Baixe os calcanhares lentamente até a posição inicial.
  • Repita para o número de repetições desejado.
×

Abdominal Máquina

  • Sente-se no equipamento com os pés firmemente apoiados no chão.
  • Segure as alavancas ou apoie o peito contra o encosto.
  • Contraia os músculos abdominais, curvando o tronco para frente.
  • Retorne lentamente à posição inicial.
  • Repita para o número de repetições desejado.
×

Bíceps Máquina

  • Sente-se no equipamento com as costas apoiadas no encosto.
  • Segure as alavancas com as palmas das mãos voltadas para cima.
  • Flexione os cotovelos, levantando as alavancas até a altura dos ombros.
  • Retorne lentamente à posição inicial.
  • Repita para o número de repetições desejado.
×

Tríceps Máquina

  • Sente-se no equipamento com as costas apoiadas no encosto.
  • Segure as alavancas com as palmas das mãos voltadas para baixo.
  • Empurre as alavancas para baixo até que os braços estejam completamente estendidos.
  • Retorne lentamente à posição inicial.
  • Repita para o número de repetições desejado.
